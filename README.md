# 🎮 Rock Paper Scissors Lizard Spock

A fully featured terminal-based Rock Paper Scissors Lizard Spock game built in C as my 1st Semester project at Hamdard University.

## 🕹️ Features

- 👤 Multi-user registration and login system
- 🔒 Password masking during input
- 💾 Persistent data storage using file handling
- 🏆 High score tracking per user
- 📊 Last 5 games history per user
- 🤖 Computer vs Computer simulation mode
- 🎨 Console color customization

## 🛠️ Built With

- C Programming Language
- File Handling
- Structs and Arrays

## 🚀 How to Run

1. Clone the repository
2. Compile using GCC: `gcc game.c -o game`
3. Run: `./game`
